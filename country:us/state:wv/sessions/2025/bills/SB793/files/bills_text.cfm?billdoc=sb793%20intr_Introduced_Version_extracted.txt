Red = new code to be enacted WEST VIRGINIA LEGISLATURE2025 REGULAR SESSION FISCAL NOTE IntroducedSenate Bill 793By Senator Maynard[Introduced March 17, 2025; referredto the Committee on Government Organization; and then to the Committee on Finance]A BILL to amend the Code of West Virginia, 1931, as amended, by adding a new section, designated §5A-3-52a, relating to creating reporting requirements for the use of aircraft by state agencies.Be it enacted by the Legislature of West Virginia: ARTICLE 3. Purchasing Division. §5A-3-52a. Required reports for state-owned aircraft use. (a) All state agencies, with the exception of the West Virginia State Police, must submit a Flight Request Form to the Aviation Division for the use of state-owned aircraft and the purpose of the flight must be certified before use is allowed. (b) The Aviation Division is required to publish a log of all flights on the Aviation Division's website with pertinent information, including the purpose of each flight. This log shall be updated at least every four weeks.NOTE: The purpose of this bill is to create reporting requirements for the use state-owned aircraft.
